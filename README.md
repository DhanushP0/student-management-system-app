## Requirements

- Flutter SDK (Latest Version)
- Dart SDK (Latest Version)
- Android Studio / VS Code with Flutter extensions

## Getting Started

1. Clone the repository
```bash
git clone [your-repository-url]
cd student_management_app
```

2. Install dependencies
```bash
flutter pub get
```

3. Run the app
```bash
flutter run
```

## Project Structure
